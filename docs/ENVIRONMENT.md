# Environment Configuration

## Setup

```bash
cp .env.example .env
php artisan key:generate
```

## Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `APP_NAME` | `ARGUX` | Application name |
| `APP_ENV` | `local` | Environment (local, staging, production) |
| `APP_KEY` | — | Auto-generated by `key:generate` |
| `APP_DEBUG` | `true` | Debug mode |
| `APP_URL` | `http://localhost:8000` | Base URL |
| `APP_LOCALE` | `en` | Default locale |
| `APP_AVAILABLE_LOCALES` | `en,hr` | Comma-separated available locales |
| `SESSION_DRIVER` | `file` | Session storage (file-based, no database) |
| `CACHE_DRIVER` | `file` | Cache driver |
| `LOG_CHANNEL` | `stack` | Logging channel |

## No Database Variables

The following are **not needed** and should not be configured:
- `DB_CONNECTION`, `DB_HOST`, `DB_PORT`, `DB_DATABASE`, `DB_USERNAME`, `DB_PASSWORD`
- `REDIS_*`
- `MAIL_*`
- `AWS_*`

## Production Notes

For production deployment, set:
```
APP_ENV=production
APP_DEBUG=false
APP_URL=https://your-domain.com
```
