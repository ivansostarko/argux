# Git

### Git Config
```bash
 git config --global user.name "Your Name"
 git config --global user.email you@example.com
 git commit --amend --reset-author
```

### Git Commit
```bash
git add --all
git commit -m ""
```

### Git Commit
```bash
git push origin HEAD
git tag -a v0.4.0 -m "v0.4.0"
git push origin v0.4.0
```
